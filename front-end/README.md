# Product Management Frontend

React + TypeScript + Vite project for Product Management System

## Getting Started

### Install Dependencies
```bash
npm install
```

### Development Server
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```

### Preview Production Build
```bash
npm run preview
```

## Tech Stack

- React 18
- TypeScript
- Vite
- ESLint
